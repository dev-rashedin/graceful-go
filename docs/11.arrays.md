# Go Arrays — Beginner-Friendly Guide

Arrays in Go are **fixed-size sequences of elements of the same type**. They are useful when you know the number of elements beforehand.

## 1. Key Features of Arrays

- **Fixed Size:** The size of an array is defined at declaration and cannot be changed
- **Type-Specific:** All elements must be of the **same type**
- **Contiguous Memory:** Elements are stored in consecutive memory locations → **fast access**
- **Zero Indexing:** The first element has index `0`

## 2. Declaring Arrays

```go
var balance [10]float32
// Declares an array of 10 float32 elements, all initialized to 0
```

- Syntax: `var arrayName [size]type`

### 2A. Initialize at Declaration

```go
var newBalance = [5]float32{100.00, 2.00, 3.00, 4.00, 5.00}
// Array of size 5 with values
```

### 2B. Initialize Without Size

```go
var newBalance2 = []float32{100.00, 2.00, 3.00, 4.00, 5.00}
// Compiler determines the size automatically
```

## 3. Accessing and Modifying Array Elements

```go
var n [10]int

// Assign values using a loop
for i := 0; i < 10; i++ {
    n[i] = i + 100
}

fmt.Println("Array Elements:", n)
// Output: Array Elements: [100 101 102 103 104 105 106 107 108 109]
```

- Access elements with **index**: `array[index]`
- Modify elements similarly: `array[3] = 999`

## 4. Iterating Through Arrays

```go
for i := 0; i < 10; i++ {
    fmt.Printf("Element[%d] = %d\n", i, n[i])
}
```

**Output:**

```
Element[0] = 100
Element[1] = 101
Element[2] = 102
Element[3] = 103
Element[4] = 104
Element[5] = 105
Element[6] = 106
Element[7] = 107
Element[8] = 108
Element[9] = 109
```

- Use **loops** to access or modify elements efficiently

## 5. Notes

- Arrays in Go have **fixed size**; use **slices** for dynamic arrays
- Arrays are **value types**, which means **assigning one array to another copies all elements**
- Use **`len(array)`** to get the size of the array

```go
fmt.Println(len(n)) // Output: 10
```
