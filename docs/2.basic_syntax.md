# Basic Syntax

## 1. Package Declaration

* Every Go program starts with a **package declaration**.
* `package main` indicates that this file is an executable program.

```go
package main
```

## 2. Importing Packages

* Use `import` to include packages that provide extra functionality.
* `fmt` is a built-in package for formatted input and output.

```go
import "fmt"
```

## 3. Comments

* **Single-line comment** starts with `//`.
* **Multi-line comment** is enclosed in `/* */`.

```go
// This is a single-line comment
/*
This is a
multi-line comment
*/
```

## 4. Main Function

* The **entry point** of a Go program is the `main` function.
* Code inside `main()` runs when the program starts.

```go
func main() {
    fmt.Println("Hello World!")
}
```

### 4.1 Printing Output

* `fmt.Println()` prints text followed by a **newline**.
* `fmt.Print()` prints text **without newline**.
* `fmt.Printf()` allows **formatted output**, similar to `printf` in C.

```go
name := "Arya"
age := 18
fmt.Println("Hello World")            // prints text with newline
fmt.Print("Hello ")                   // prints text without newline
fmt.Printf("Name: %s, Age: %d\n", name, age)  // formatted output
```

* `%s` → string placeholder
* `%d` → integer placeholder
* `\n` → newline character
